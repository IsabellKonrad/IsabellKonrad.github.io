Songs of God

Songs of God ist eine Web-App mit über 200 christlichen Lieder mit Gitarrenbegleitung. Beim Hinzufügen von neuen Liedern zur Liedersammlung wird die Tonart des Liedes automatisch mithilfe einer Support Vector Machine erkannt, welche in Python mit Scikit-learn geschrieben ist. Jedes Lied kann in jede Tonart transponiert und als PDF abgerufen werden, wobei eine RESTFUL API in Flask und Python verwendet wird.

Read Notes

Im Projekt Read Notes wird ein Klassifizierer erstellt, der erkennt ob Bilder mit Notenzeilen Noten enthalten oder nicht. Zum Bauen des Klassifizierers wird die Machine Learning Bibliothek TensorFlow verwendet. In mehreren Schritten werden eingescannte Musikstücke zunächst in einzelne Notenzeilen, dann in die einzelnen Noten oder nicht-Noten zertrennt. Nicht-Noten sind zum Beispiel Notenschlüssel, Pausen oder Taktstriche. Momentan hat das neuronale Netz drei sogenannte hidden layers und erkennt Noten mit einer Genauigkeit von 80\%.
Code auf Github

Math Exam Recources

Zur Verbesserung der Lernmöglichkeiten der Mathematik-Studenten an der University of British Columbia werden Mathematik-Altklausuren gelöst und auf einer online-Lernplattform mit MediaWiki veröffentlicht. Die Studenten finden dort klausurrelevante Aufgaben, Tipps und geprüfte Lösungen, können Bemerkungen und Bewertungen abgeben, und es gibt Links zu ähnlichen Aufgaben und Lernmaterial. Nach Entwicklung seit 2012 am MER sind über 1500 gelösten Fragen online. Unsere Wiki-Seiten werden über 400.000 mal pro Semester angeklickt.
www.math-exam-resources.com

Windrad-Abstand

Seit Mitte des Jahre 2013 wird darüber diskutiert, ob bei der Bauung von Windrädern in Bayern die sogenannte 10H-Idee eine sinnvolle Regelung sei. Die 10H-Idee besagt, dass der Abstand von einem Windrad zum nächsten Wohngebiet mindestens die zehnfache Höhe des Windrades betragen muss. Eine dynamische, bequeme Übersicht der Flächen von Unterfranken, die aufgrund dieser Regelung zur Bauung von Windrädern noch verfügbar sind, findet man auf folgender Webseite.

Würfelspiel

Das Würfelspiel ist ein lustiges, belebendes Spiel mit fünf Würfeln, das alleine oder mit beliebig vielen Personen gespielt werden kann. Eine Matlab-Version als Computerspiel kann man sich unter folgendem Link von der Mathworks-Webseite File Exchange herunterladen (Dice Game by Bernhard Konrad). Bei dem Computerspiel kann man sowohl gegen menschliche Gegner als auch gegen unterschiedlich starke Computer-Gegner spielen.
Die Computer-Gegner wurden mithilfe eines Würfelspiel-Wettbewerbs (Dice Game Contest) von vier Teilnehmern entwickelt.

Viria

Viria ist ein informatives, strategisches Brettspiel. Thematisch geht es um die Dynamiken im Körper eines Menschen, der sich mit dem HI-Virus infiziert hat. Das Spiel soll dazu dienen, die komplizierten Vorgänge und die unterschiedlichen "Strategien" der HI-Viren zu verstehen, indem diese vereinfacht und bildhaft im Brettspiel wiedergegeben werden.
